# My Documents

**New in v1.2.** Save the worksheets you care about and come back to them any time. Build a library of spelling lists, sight-word sets, and custom passages — each one remembers its own grade level, font, and other settings.

## Opening the Library

Tap the **gray folder button** in the Speak step. It's always visible when you're idle — you'll find it alongside the microphone, keyboard, and templates (orange) buttons.

The folder button is hidden while you're actively recording so it doesn't get in the way.

## Saving a Document

With some text in the transcript area, tap the folder button and then:

- **Save Current Text…** — saves a brand-new document with a title you choose
- **Update "\<title\>"** — replaces the loaded document with your latest edits (only shown when a saved document is already loaded)
- **Save as Copy…** — forks the loaded document into a new one, leaving the original untouched

You'll be prompted for a **title** (the app suggests one from your first few words, but you can type anything) and a **toggle** to include the worksheet settings.

### Include worksheet settings

When this toggle is on, the document remembers every setting on the worksheet:

- Grade level
- Font style
- Letter darkness
- Letter spacing
- Cursor width
- Rendering style (Solid Fill or Dotted Outline)

This is the right choice for most classroom use — a teacher can have one document per student, each with the right grade level and font, and reloading pulls the whole setup back.

Leave it off if you just want the **text** saved and don't care about the visual style.

## Loading a Saved Document

Tap the folder button and pick any row from the list. The text drops into the Speak step and any saved settings are restored. A small **document badge** appears above the transcript area so you always know which saved document you're editing.

:::tip
When a document is loaded, editing the text and then tapping **Update "\<title\>"** saves your edits back into the same file. If you want to keep both the original and the edited version, use **Save as Copy…** instead.
:::

## Renaming and Deleting

In the list:

- **Swipe left** on a row to reveal **Delete** and **Rename** buttons
- **Long-press** (or right-click with an external keyboard) to open a context menu with the same options

Deleting a document is permanent — there's no undo for now. If the document you delete is the one currently loaded, the document badge disappears but **your text stays in the editor** so you don't lose your work in progress.

## Starting Fresh

To close the current saved document and start a brand-new draft:

- Tap the folder button, then **New Document**
- Or use the toolbar's **Start Over** button (which also clears the PDF preview)

The text in the editor is preserved either way — only the "currently editing X" marker is cleared. Use the **X (clear)** button if you want to empty the editor too.

## Where Your Documents Live

Saved documents live on your iPad in the app's private storage — they're included in your iCloud **device backup** if you have iCloud Backup turned on, but they aren't synced across iPads.

:::note
Cross-iPad sync is planned for a future release. For now, if you use AtlasTraceWrite on multiple iPads, each one maintains its own library.
:::
